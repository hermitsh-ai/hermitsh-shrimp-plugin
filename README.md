# Hermit Shell — Claude Plugins

Plugins by [Hermit Shell](https://hermitsh.ai) for Claude.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **shrimp-tasks** | Connect Claude to SHrimp — read, create, and manage your tasks from chat |

## Install

Add this marketplace in Claude:

```
/plugin marketplace add hermitsh/claude-plugins
```

Then install a plugin:

```
/plugin install shrimp-tasks@hermitsh
```

## Setup (SHrimp Tasks)

1. Install the plugin (above)
2. Open SHrimp on your phone → Settings → Agents → Pair New Agent
3. Tell Claude the 6-digit code
4. Done — Claude can now manage your tasks
